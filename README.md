# 人生を経営するための行動管理システム Backend

## 概要

人生を経営するための行動管理システムのバックエンドAPIです。

Django REST Frameworkを利用し、

* 営業
* 開発
* 健康
* 計画

の4つの視点から管理するタスクデータを提供します。

Reactフロントエンドと連携し、タスクの登録・取得・更新・削除を行うためのREST APIを構築しています。

---

## このプロジェクトの目的

このAPIは単なるToDoアプリのバックエンドではありません。

**「人生を経営するための行動管理システム」**

として、

* 営業
* 開発
* 健康
* 計画

を継続的に管理できる基盤を構築することを目的としています。

---

## API一覧

### タスク一覧取得

```http
GET /api/tasks/
```

登録済みタスクを取得します。

---

### タスク詳細取得

```http
GET /api/tasks/{id}/
```

指定したタスクを取得します。

---

### タスク登録

```http
POST /api/tasks/
```

新しいタスクを登録します。

---

### タスク更新

```http
PUT /api/tasks/{id}/
PATCH /api/tasks/{id}/
```

タスクを更新します。

---

### タスク削除

```http
DELETE /api/tasks/{id}/
```

タスクを削除します。

---

## データモデル

### Task

| 項目          | 型       | 説明   |
| ----------- | ------- | ---- |
| id          | Integer | ID   |
| title       | String  | タスク名 |
| description | Text    | 詳細説明 |
| category    | String  | カテゴリ |
| completed   | Boolean | 完了状態 |
| deadline    | Date    | 期限   |
