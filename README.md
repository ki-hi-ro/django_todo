# 資産化期の行動管理システム Backend

## 人生を経営するためのタスク管理システム

Django REST Frameworkで構築したバックエンドAPIです。

収入・技術資産・回復・戦略の4つのカテゴリを管理し、資産形成につながる行動を可視化することを目的としています。

---
